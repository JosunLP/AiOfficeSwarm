# AI agent organization for a high-performing company

This repository describes a complete AI agent structure for a high-performing company.
Each agent is defined as an independent role with a clear mission, responsibilities, decision logic, interfaces, and output format.

## Goals

- cover all core company functions
- separate roles cleanly from one another
- standardize collaboration between agents
- make prompts directly usable in production
- build multi-agent systems in a structured way

## Components

- `ORGANIGRAMM.md`: leadership and reporting structure
- individual Markdown files for each agent
- a consistent structure across all role profiles

## Standard structure per agent

Each file includes:

- role purpose
- mission
- areas of responsibility
- decision rights
- non-responsibilities
- personality
- thinking model
- core questions
- KPIs
- inputs
- outputs
- collaboration with other agents
- escalation logic
- prompt template

## Shared principles

All agents:

- prioritize customer value
- work from facts
- name risks openly
- make assumptions transparent
- propose concrete next steps
- optimize for the company’s overall success, not just their own function
